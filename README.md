# Muhammad Taha Rajpoot — Portfolio 💻🛡️

> Cybersecurity Expert · Penetration Tester · CS Engineer

A modern, dark-themed portfolio showcasing my cybersecurity expertise, professional certifications, security tools, and career journey. Built with React.js and a unified glassmorphic design system.

🔗 **Live:** [muhammad-taharana.vercel.app](https://muhammad-taharana.vercel.app/)

---

## 🌟 Sections

| Section | Description |
|---------|-------------|
| **Hero** | Interactive spotlight image, CV download/view buttons, professional tagline |
| **About** | Bio from CV, TeamCyberØps membership, flip photo card, stats |
| **Projects** | 8 security tools from GitHub with descriptions and tech tags |
| **Certifications** | 5 professional certs (C\|EH, RHCE, GCPDE, ISCS, PT) with PDF viewer |
| **Services** | Vulnerability Assessment, Pentesting, Bug Bounty, Recon, Tool Dev |
| **Contact** | Contact form, email, phone, location, social links |

---

## ⚡ Key Features

- 🎨 **Unified Dark Glassmorphic Theme** — Consistent design across all sections
- 📄 **CV Integration** — Download or view CV inline with smart document viewer
- 🏆 **Certificate Viewer** — View PDF certificates directly in the browser
- 🔄 **Interactive Elements** — Spotlight image, flip cards, hover animations
- 📱 **Fully Responsive** — Desktop, tablet, and mobile optimized
- 🎯 **SEO Optimized** — Meta tags, OG tags, semantic HTML
- ⌨️ **Keyboard Support** — ESC to close modals

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/muhammadtaharana/Taha-Portfolio.git
cd Taha-Portfolio

# Install
npm install

# Dev server
npm run dev

# Build
npm run build
```
